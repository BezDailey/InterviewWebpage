# Interview For CBC
    The Interview Website is in ordered modules.

## Modules
1. Start Up Module
1. HTML Module
1. CSS Module
1. JS Module

## Important Focuses

- Beginner friendly
- Tests HTML, CSS, and JS
- Set objectives and give resources to help 
- 2 hours

## Module Content
### Start Up Module
- Folder Creation
- Opening Visual Studio Code
### HTML Module
- Header element
  - H1 tag *name*
- Article element 
  - p1 tag *stuff about you* 
  - button
- Table
  - Favorite place, food, and tv show
- Footer element
  - Link to the database
### CSS Module
- Layout
  - Floats
  - 2 Column
- Style
  - Change font
  - Add background img
  - Change text color
### JS Module 
- HTML Event Handler for button
- Function that creates an alert
  - Variables
    - Name
    - Age
  - Output
    - Alert of name and age
